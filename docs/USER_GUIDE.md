# TMD Quasar App User Guide

Welcome to the Tango Marathons & Festivals (TMD) app! This guide will help you navigate and make the most of its features.

## Introduction

This application provides a comprehensive listing of tango events like marathons, festivals, and encuentros, as well as profiles for DJs and teachers. You can easily find events, learn about DJs, and explore the tango world.

## Navigating the App

The main navigation bar (usually at the top or in a side menu on mobile) will typically include sections like:

-   **Events / Marathons / Festivals / Encuentros**: These sections list tango events. They might be grouped or have a unified "Events" view.
-   **DJs**: Browse profiles of tango DJs.
-   **Teachers**: Find information about tango teachers.
-   **About**: Information about the application or website.
-   **Contact**: How to get in touch.

## Finding Events

The core feature of the app is its event listing. Here's how to use it effectively:

### 1. Event Lists

When you navigate to an event section, you'll see a list of events. These might be displayed as cards or in a table. Each event entry will typically show:
-   Event Name
-   Dates
-   Location (City, Country)
-   Event Type/Categories (e.g., Marathon, Festival)

### 2. Filtering Events

To find specific events, use the available filters. These are usually located near the top of the event list page or in a collapsible panel. Common filters include:

-   **Search**: A search bar to type keywords (e.g., event name, city).
-   **Country**: Select a country to see events only in that location.
-   **Date Ranges**:
    -   **Event Dates**: Specify a start and end date for when the event occurs.
    -   **Registration Dates**: Filter by when registration opens or closes (if available).
    -   **Quick Filters**: Options like "Next 4 months" or "Next 9 months" for convenience.
-   **Event Category**: Filter by type, such as Marathons, Festivals, Encuentros, Workshops, etc. (These might be color-coded for easy identification).
-   **Past Events**: A toggle or checkbox to include or exclude events that have already happened.

**How to Use Filters:**
1.  Locate the filter controls.
2.  Make your selections (e.g., choose a country from a dropdown, pick dates from a calendar).
3.  The event list should update automatically to reflect your filter choices.
4.  Your filter preferences may be saved (e.g., in cookies) for your next visit.

### 3. Viewing Event Details

Clicking on an event in the list will take you to its detail page. Here you'll find more comprehensive information, such as:
-   Full event description
-   Organizer details (name, email, website)
-   Venue information
-   List of DJs playing at the event
-   List of teachers (if applicable)
-   Registration link and pricing
-   Schedule (if available)

On mobile, you might be able to "pull-to-refresh" on the event details page to get the latest information.

## Exploring DJ Profiles

The "DJs" section allows you to browse and view profiles of tango DJs.

### 1. DJ List

Similar to events, DJs will be listed, showing their name and perhaps country or city. You can often search or filter this list.

### 2. DJ Details Page

Clicking on a DJ's name opens their profile, which may include:
-   DJ Name and Photo
-   Biography or "About Me" section
-   Country and City
-   Contact information (email, website, Facebook links)
-   Activity details (e.g., whether they DJ at marathons, festivals, etc., and since when)
-   A list of events they have played at or are scheduled to play at.

## Exploring Teacher Profiles

The "Teachers" section works similarly to the DJ section:
-   A list of teachers.
-   Detailed teacher profiles with information about their teaching style, experience, and events they are associated with.

## Event Series

Some events might be part of a series (e.g., an annual event). The app may allow you to view events grouped by series.

## Mobile Usage

The application is designed to be mobile-first:
-   **Responsive Layout**: The layout adapts to your screen size.
-   **Touch-Friendly**: Controls are optimized for touch interaction.
-   **Collapsible Filters**: On smaller screens, filters are often in a panel that can be opened and closed to save space.
-   **Pull-to-Refresh**: On some lists or detail pages, you can pull down from the top of the screen to refresh the content.

## Dark Mode

The application may support a dark mode for comfortable viewing in low-light conditions. This might be a manual toggle in the settings or follow your system's preference.

## Tips for Effective Use

-   **Use Filters**: The best way to find what you're looking for is to use the filter options.
-   **Check Event Details**: Always open the event details page for the most complete information before making plans.
-   **Look for Embedded Info**: DJ profiles might list their events, and event pages might list their DJs. This helps you discover related content.

We hope this guide helps you enjoy using the TMD app!
